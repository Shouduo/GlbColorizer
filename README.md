# GlbColorizer

'GlbColorizer' is simply a intuitive colorizer for .glb file. To preview every color combination easily before production, just try as many as you wish.

## Project setup

``` shell
git clone https://github.com/Shouduo/GlbColorizer.git
cd GlbColorizer
yarn
yarn start
```
